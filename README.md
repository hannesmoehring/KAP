# Klinisches Anwendungsprojekt - Neurologiedaten

## How to run

### 1. Create a virtual environment 

`python3 -m venv .venv`

### 2. Activate virtual environment

On Linux/MacOS:
`source .venv/bin/activate`

On Windows:
`.venv/bin/activate`

### 3. Install dependencies
`pip install -r requirements.txt`

### 4. Start program
`python3 main.py`
